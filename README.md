# 机器学习入门课程
腾讯云学院提供的机器学习入门课程，适合机器学习入门者，或对机器学习感兴趣的人进行学习。

## 课程简介
本课程将从实际应用出发，构建机器学习在实际工程应用中的整体学习路线。

本课程将对机器学习的整体应用流程、几类基本问题的解决方案进行详实的介绍，并应用实际例子引导初学者入门机器学习，并付诸于实践。

## 课程目录

1. 整体概述及基础知识准备
2. 特征工程与数据可视化
3. 模型与算法： 聚类模型
4. 模型与算法： 分类模型
5. 模型与算法： 回归模型
6. 模型与算法： 关联挖掘模型
7. 模型评估与算法调优
